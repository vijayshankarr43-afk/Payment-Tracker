#Supplier Payment Processing Tracker

## About This Project
A Microsoft Excel project simulating supplier 
and employee payment operations — modelled on 
 Finance team workflows.

## What It Includes
- 40 mock supplier & employee payments
- Failed payments exception report
- VLOOKUP-based supplier lookup tool
- KPI dashboard with charts

## Skills Demonstrated
- Microsoft Excel — Advanced
- VLOOKUP & SUMIF formulas
- Payment processing workflow
- Exception reporting & escalation
- Data analysis & visualization

## Tools Used
- Microsoft Excel
- VLOOKUP, COUNTIF, SUMIF, IFERROR

## Project Purpose
Built to demonstrate practical finance 
operations skills for Payments Specialist 
roles in financial services.
